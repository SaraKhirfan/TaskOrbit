# TaskOrbit – Scrum Methodology Project Management Mobile Application System

TaskOrbit is a mobile application designed to help software teams manage their projects using Scrum methodology. It supports backlog management, sprint planning, daily task tracking, and progress monitoring in a simple and organized workflow.

This application was developed as **My Graduation Project** at **the University of Jordan**, completed in **June 2025**, and received a **grade of A (4/4)**.

---

## Overview

TaskOrbit provides a structured way for teams to implement Scrum. It organizes work into backlogs and sprints, and allows users to follow task progress throughout the sprint lifecycle. The goal is to provide clarity, improve team coordination, and maintain visibility over project status.

---

## Features

### Product Backlog
- Add, edit, and delete backlog items  
- Assign priority and category  
- Organize items before sprint planning  

### Sprint Planning
- Create sprints with start and end dates  
- Move backlog items into sprints  
- Assign workload and responsibilities  

### Task Tracking
- Update task status (To Do, In Progress, Completed)  
- Add notes and descriptions  
- Daily progress visibility  

### Reporting
- Sprint progress overview  
- Completed vs remaining tasks  
- Basic performance insights  

### User Management
- User registration and login  
- Profile handling  
- Individual task assignments
